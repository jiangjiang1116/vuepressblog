​ 最近修改了页面的一个样式，为了加强对样式的学习和对付比较典型的需求，我寻找到了一个低代码平台的项目：JEECG官方网站Github地址

​ 另外，我还发现了几个样式站：Viser (viserjs.github.io)AntV | 蚂蚁数据可视化 (antgroup.com)

项目下载和运行

  • 拉取项目代码
git clone https://github.com/zhangdaiscott/jeecg-boot.git
cd  jeecg-boot/ant-design-vue-jeecg
  • 安装依赖
yarn install
  • 开发模式运行
yarn run serve
  • 编译项目
yarn run build
  • Lints and fixes files
yarn run lint

Docker镜像启动前端（单体模式）

# 1.配置host

   127.0.0.1   jeecg-boot-system
   
   这里在vue.config.js里修改，把target: 'http://localhost:8080',改为  target: 'http://127.0.0.1:8080'

# 2.修改前端项目的后台域名
   .env.development里的
   VUE_APP_API_BASE_URL=http://localhost:8080/jeecg-boot

   域名改成： http://jeecg-boot-system:8080/jeecg-boot
   
   
  
# 3.进入项目根目录，执行打包命令
 yarn run build

# 4.构建镜像
 docker build -t jeecgboot-ui2 .
 
 停在这步，来活了

# 5.启动镜像
 docker run --name jeecgboot-ui-vue2 -p 80:80 -d jeecgboot-ui2

# 6.访问前台项目
 http://localhost

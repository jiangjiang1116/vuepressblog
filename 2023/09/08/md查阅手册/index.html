    编写 Markdown 文章的基本语法和示例：

    标题：

    使用 # 符号来表示标题，# 的数量表示标题级别，例如：

    # 一级标题
## 二级标题
### 三级标题

    段落：

    段落之间用空行分隔，可以在段落中直接输入文本，不需要额外的标记。

    列表：

    可以使用 *、- 或 1. 2. 3. 来创建无序或有序列表，例如：

    - 无序列表项 1
- 无序列表项 2
  - 嵌套的无序列表项
1. 有序列表项 1
2. 有序列表项 2

    链接：

    [OpenAI](https://www.openai.com/)
//[链接文本] (链接地址)

    图片：

    使用 来插入图片，例如：

    ![示例图片](https://example.com/image.jpg)

    引用：

    使用 > 来创建引用，例如：

    > 这是一个引用文本。
    预览

    这是一个引用文本。

    粗体和斜体：

    使用 粗体 和 斜体 来表示粗体和斜体文本，例如：

    **这是粗体文本**
*这是斜体文本*
    预览

    这是粗体文本
    这是斜体文本

    代码块：

    使用三个反引号 ` 或四个空格来创建代码块，例如：

    ```
function helloWorld() {
    console.log("Hello, World!");
}
```
    指定代码块语言

    使用三个反引号（```）创建代码块，可以在后面指定语言（可选）：

    ```javascript
function helloWorld() {
    console.log("Hello, World!");
}

    你可以在反引号后面添加编程语言的名称，以便正确高亮显示代码，例如：``javascript`

    使用四个空格缩进来创建代码块
        function helloWorld() {
        console.log("Hello, World!");
    }

    分割线：

    使用三个或更多的 -、* 或 _ 来创建水平分割线，例如：

    ---

    表格：

    创建表格需要使用 | 来分隔单元格，例如：

    | 表头1 | 表头2 |
| ------ | ------ |
| 单元格1 | 单元格2 |
    预览：
    表头1 表头2
    单元格1 单元格2
    其他方法

    使用Typora等工具，提供了一个直观的图形用户界面（GUI），使用户可以轻松地编辑和预览 Markdown 格式的文档。
